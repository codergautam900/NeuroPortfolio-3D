## рҹҢҢ Features

<br/>

### рҹҺ® 3D & Visuals
- **Real-time 3D Scenes** вҖ” Built with React Three Fiber on top of Three.js, achieving GPU-accelerated rendering directly in the browser
- **Physics Simulation** вҖ” Rigid body & collider physics via `@react-three/rapier` (Rapier WASM engine)
- **Post-Processing FX** вҖ” Bloom, depth-of-field, vignette, and more via `@react-three/postprocessing`
- **Cinematic Camera Work** вҖ” Controlled camera paths and orbital transitions for a film-quality feel

### рҹӨ– AI & Intelligence
- **AI Oracle** вҖ” An embedded streaming chat endpoint (`/api/oracle`) trained with context about the developer, letting visitors ask anything
- **AI Tagline Generator** вҖ” Unique, creative taglines generated via OpenAI on every visit, using `/api/tagline`
- **Vercel AI SDK** вҖ” Streaming responses, structured output, and OpenAI integration with minimal boilerplate

### рҹҺһпёҸ Motion & Scroll
- **Scroll-Driven Storytelling** вҖ” GSAP ScrollTrigger drives 3D scene transitions as users scroll
- **Framer Motion** вҖ” Declarative React animations for UI elements, page transitions, and micro-interactions
- **Lenis Smooth Scroll** вҖ” Buttery-smooth, momentum-based scrolling for a premium feel

### рҹ’Ң Contact & Communication
- **Contact Form** вҖ” Fully functional email delivery via **Resend** + **React Email** components
- **Zod Validation** вҖ” Server-side input validation to keep everything safe and type-correct

### рҹҺЁ Design & UX
- **Responsive Design** вҖ” Flawlessly adapts across all screen sizes from mobile to ultrawide
- **Optimized Typography** вҖ” **Syne**, **Manrope**, and **Space Mono** served locally via `@fontsource`
- **Utility-First Styling** вҖ” Tailwind CSS v4 with `tailwind-merge` and `clsx` for clean conditional classes
- **Lucide Icons** вҖ” A lean, consistent icon set across the entire UI

## рҹӨ– AI Features

### рҹ”® Oracle вҖ” `/api/oracle`

An intelligent, streaming chat endpoint powered by the **Vercel AI SDK** and **OpenAI**. The Oracle is given a system prompt with rich context about the developer вҖ” projects, skills, experience, personality. Visitors can ask it anything and get instant, streamed responses.

**How it works:**
```
User message вҶ’ POST /api/oracle вҶ’ OpenAI (stream) вҶ’ Response chunks вҶ’ UI
```

The response is streamed back token-by-token for a fast, conversational feel.

### вңҚпёҸ Tagline Generator вҖ” `/api/tagline`

Calls OpenAI on every visit to generate a fresh, creative tagline so the portfolio never feels stale. Taglines reflect the developer's personality and are constrained to be punchy and impactful.

**How it works:**
```
Page load вҶ’ GET /api/tagline вҶ’ OpenAI вҶ’ unique tagline вҶ’ Rendered in hero
```

## рҹҺ¬ Animations & 3D Pipeline

### Scroll-Driven 3D with GSAP
GSAP's `ScrollTrigger` plugin pins sections and orchestrates 3D camera movements, object animations, and scene transitions as the user scrolls. This creates a **narrative-driven experience** where each scroll position tells part of the story.

### Physics with Rapier
`@react-three/rapier` wraps the Rapier WASM physics engine, enabling real-time rigid body simulation, colliders, and gravity effects вҖ” all running at 60fps in the browser.

### Post-Processing Stack
The rendering pipeline applies effects on top of the Three.js scene using `@react-three/postprocessing`:
- **Bloom** вҖ” glowing highlights for lights and emissive materials
- **Depth of Field** вҖ” cinematic focus blur for depth
- **Vignette** вҖ” subtle dark edges for a cinematic frame

### Lenis + Framer Motion
Lenis intercepts native scroll events to provide momentum-based smooth scrolling, which feeds into GSAP's ScrollTrigger. Framer Motion handles all React UI animations, from page entry transitions to hover states and modal reveals.

## рҹҡў Deployment

### Vercel (Recommended)

The fastest path from code to production:

1. Push your code to GitHub
2. Go to [vercel.com/new](https://vercel.com/new) and import your repository
3. Add environment variables in the **Environment Variables** section:
   - `OPENAI_API_KEY`
   - `RESEND_API_KEY`
4. Click **Deploy** рҹҡҖ

> [!TIP]
> Vercel auto-detects Next.js, handles incremental builds, and gives you a globally distributed CDN вҖ” perfect for a portfolio that needs to load fast everywhere.

### Self-Hosted / Other Platforms

```bash
# Build the production bundle
npm run build

# Start the production server
npm run start
```

Make sure your platform supports **Node.js вүҘ 18** and set the environment variables in your hosting dashboard.

> [!IMPORTANT]
> Rapier uses WebAssembly. Ensure your CDN/host serves `.wasm` files with the correct `Content-Type: application/wasm` header. Vercel handles this automatically.

- **3D model optimization** вҖ” Use tools like [gltf-transform](https://gltf-transform.donmccurdy.com/) or Blender to compress GLTF models before adding them to `/public/models/`
- **Texture compression** вҖ” Use `.ktx2` (Basis Universal) textures with drei's `useKTX2` for best GPU performance
- **Lazy-load 3D scenes** вҖ” Use `React.lazy` + `Suspense` around heavy canvas components so they don't block the initial paint
- **Reduce draw calls** вҖ” Merge static geometries and use instanced meshes for repeated objects
- **Environment maps** вҖ” Use compressed `.hdr` files at 512px resolution for environment lighting
